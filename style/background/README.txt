== HƯỚNG DẪN ĐẶT FILE BACKGROUND ==

Đặt một trong các file sau vào thư mục này để thay background trang web:

  bg.mp4   → Video nền (khuyến nghị, hỗ trợ tốt nhất)
  bg.webm  → Video nền (định dạng thay thế)
  bg.gif   → Ảnh động GIF
  bg.png   → Ảnh tĩnh PNG
  bg.jpg   → Ảnh tĩnh JPG
  bg.jpeg  → Ảnh tĩnh JPG
  bg.webp  → Ảnh tĩnh WebP

Lưu ý:
- Chỉ đặt 1 file tại một thời điểm (ưu tiên theo thứ tự trên nếu có nhiều file)
- Video sẽ tự phát không tiếng, lặp vô hạn
- Nếu không có file nào → trang dùng gradient hồng mặc định
- Nên dùng video độ phân giải vừa phải (720p hoặc 1080p) để tải nhanh
